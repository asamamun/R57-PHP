Indexing

An index is an ordered (or indexed) subset of table columns, with each row entry pointing to its corresponding table row.
advantages of Indexing:

  • Query optimization
  • Uniqueness
  • Text searching

These advantages are realized, thanks to four types of indexes:

  • primary
  • unique :Like a primary index, a unique index prevents duplicate values from being created.However, the difference is that only one primary index is allowed per table, whereas multiple unique indexes are supported.
    ALTER TABLE `marks` ADD UNIQUE `student_quiz`(`stid`, `quizid`);
  • normal
  • full-text : SELECT * FROM film WHERE MATCH(description) AGAINST ('fight');
